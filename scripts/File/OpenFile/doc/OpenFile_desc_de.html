Benutzen Sie dieses Kommando um Zeichnungesdateien zu öffnen. Die Zeichnung wird aus der Datei geladen und in einem neuen Fenster dargestellt. Sie können zwischen den geöffneten Zeichnungen hin und her wechseln mit dem Menü 'Fenster' oder den Karteireitern oben in QCAD Fenster.

PDF Dateien

Wenn Sie eine PDF Datei öffnen, zeigt QCAD einen Dialog an. In diesem Dialog können Sie die zu importierende Seite angeben, ob Sie Bilder importieren wollen und ob Clipping (Zuschneiden) angewendet werden soll. Importierte Bilder werden in einem neuen Ordner namens "pdf_images" gespeichert, der sich dort befindet, wo sich auch die PDF-Datei befindet. Ohne Clipping werden alle Vektordaten aus der PDF-Datei importiert. Dies kann auch Daten ausserhalb des Papiers umfassen.

CXF Schriftdateien

QCAD kann auch CXF Schriftdateien öffnen. Wenn eine solche Datei geöffnet wird, wird jede Glyphe in einen separaten Block geladen. Sie können dann die Glyphe bearbeiten oder neue Glyphen hinzufügen. Blocknamen sind im Format "[Hexadezimal Unicode] [Zeichen]", zum Beispiel "0x0041 A" für ein grosses "A" mit Unicode 65 (41 in hexadezimal).

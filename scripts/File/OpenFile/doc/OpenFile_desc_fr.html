Utilisez cette commande pour ouvrir un fichier de dessin. Le fichier est chargé et affiché dans une nouvelle fenêtre dans la fenêtre de l'application QCAD. Vous pouvez basculer entre les différents dessins chargés en utilisant le menu "Windows" ou les onglets en haut de la zone de dessin.

Fichiers PDF

Si vous ouvrez un fichier PDF, QCAD affiche un dialogue. Dans ce dialogue, vous pouvez spécifier la page à importer, si vous voulez importer des images et si le détourage doit être appliqué. Les images importées sont stockées dans un nouveau dossier appelé "pdf_images" situé à l'endroit où se trouve le PDF. Sans écrêtage, toutes les données vectorielles sont importées à partir du fichier PDF. Cela peut inclure des données situées à l'extérieur du papier.

Fichiers de police CXF

QCAD peut également ouvrir des fichiers de police CXF. Lors de l'ouverture d'un tel fichier, chaque glyphe est chargé dans un bloc séparé. Vous pouvez ensuite éditer le glyphe ou ajouter de nouveaux glyphes. Les noms de blocs sont au format "[Hexadécimal Unicode] [Caractère]", par exemple "0x0041 A" pour un "A" majuscule avec Unicode 65 (41 en hexadécimal).

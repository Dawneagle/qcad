Usare questo comando per aprire un file di disegno. Il file viene caricato e mostrato in una nuova finestra all'interno della finestra dell'applicazione QCAD. È possibile passare da un disegno caricato all'altro utilizzando il menu "Windows" o le schede nella parte superiore dell'area di disegno.

File PDF

Se si apre un file PDF, QCAD mostra una finestra di dialogo. In questa finestra di dialogo è possibile specificare la pagina da importare, se si desidera importare immagini e se deve essere applicato il ritaglio. Le immagini importate vengono memorizzate in una nuova cartella chiamata "pdf_images", situata nel punto in cui si trova il file PDF. PDF. Senza ritaglio, tutti i dati vettoriali vengono importati dal file PDF. Ciò potrebbe includere dati esterni alla carta.

File di font CXF

QCAD può anche aprire file di font CXF. Quando si apre un file di questo tipo, ogni glifo viene caricato in un blocco separato. È quindi possibile modificare il glifo o aggiungerne di nuovi. I nomi dei blocchi sono nel formato "[Unicode esadecimale] [Carattere]", ad esempio "0x0041 A" per una "A" maiuscola con Unicode 65 (41 in esadecimale).

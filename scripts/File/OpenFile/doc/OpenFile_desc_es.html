Utilice este comando para abrir un archivo de dibujo. El archivo se carga y se muestra en una nueva ventana dentro de la ventana de la aplicación QCAD. Puede cambiar entre los diferentes dibujos cargados utilizando el menú "Ventanas" o las pestañas de la parte superior del área de dibujo.

Archivos PDF

Si abre un archivo PDF, QCAD muestra un diálogo. En este diálogo, puede especificar la página a importar, si desea importar imágenes y si se debe aplicar el recorte. Las imágenes importadas se almacenan en una nueva carpeta llamada "pdf_images" situada donde se encuentra el archivo PDF. Sin recorte, todos los datos vectoriales se importan desde el archivo PDF. Esto puede incluir datos fuera del papel.

Archivos de fuentes CXF

QCAD también puede abrir archivos de fuentes CXF. Al abrir un archivo de este tipo, cada glifo se carga en un bloque separado. A continuación, puede editar el glifo o añadir nuevos glifos. Los nombres de los bloques tienen el formato "[Hexadecimal Unicode] [Carácter]", por ejemplo "0x0041 A" para una "A" mayúscula con Unicode 65 (41 en hexadecimal).

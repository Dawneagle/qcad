Use este comando para abrir um ficheiro de desenho. O ficheiro é carregado e mostrado numa nova janela dentro da janela da aplicação QCAD. Pode-se alternar entre diferentes desenhos carregados usando o menu "Windows" ou os separadores na parte superior da área de desenho.

Ficheiros PDF

Se abrir um ficheiro PDF, o QCAD mostra um diálogo. Neste diálogo, pode especificar a página a importar, se quiser importar imagens e se o recorte deve ser aplicado. As imagens importadas são armazenadas numa nova pasta chamada "pdf_images", localizada no local onde se encontra o ficheiro PDF. Sem clipping, todos os dados vectoriais são importados a partir do ficheiro PDF. Isto pode incluir dados fora do papel

Ficheiros de Fontes CXF

O QCAD também pode abrir ficheiros de fontes CXF. Ao abrir um tal ficheiro, cada glifo é carregado num bloco separado. Pode então editar o glifo ou adicionar novos glifos. Os nomes dos blocos estão no formato "[Hexadecimal Unicode] [Caracter]", por exemplo "0x0041 A" para um "A" maiúsculo com Unicode 65 (41 em hexadecimal)
